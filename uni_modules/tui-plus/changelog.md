## 4.0.0（2025-08-16）
### 新增组件
- `t-marked`：富文本渲染组件，支持 Markdown 和自定义 HTML 渲染。
- `t-marked-code`：富文本代码高亮组件，支持多种编程语言语法高亮。
- `t-marked-table`：富文本表格组件，提供灵活的表格渲染能力。
- `t-marked-td`：富文本表格单元格组件。
- `t-marked-th`：富文本表格表头组件。
- `t-marked-tr`：富文本表格行组件。
- `t-marquee`：跑马灯组件，支持多种滚动模式和自定义样式。
- `t-waterflow`：瀑布流布局组件，自适应不同屏幕尺寸。
- `t-sprite`：序列帧动画组件，支持图片序列播放。
- `t-verify-puzzle`：滑块拼图验证组件，提升安全性。
- `t-lucky-wheel`：幸运大转盘组件，支持自定义奖品和动画效果。
- `t-dropdown-menu`：下拉菜单组件。
- `t-dropdown-item`：下拉菜单子组件。
### 重构优化
- 所有组件重构为 Vue 3 Setup 语法。
- 移除不再推荐的 Vue 2 语法，提供更符合 Vue 3 的开发体验。
- 细节优化和性能提升。
